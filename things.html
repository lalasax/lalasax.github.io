Here are some tips I've learned:

  • To auto indent in sublime text 2, use this :

    • { "keys": ["`"], "command": "reindent", "args": {"single_line": false} }

    Edit that in Preferences > keybinding > user, for me I used "`" because that's the only button I rarely use :).

  • To add phonetic to excel, just add a new group called phonetic, then add all phonetic related stuff to it, after that just use edit and show phonetic, remember to download the fonts and things needed in option
  • To gettext from img file, use smartOCGLITE, to change bunch of file from png to jpg, use Batch png to jpg
  • To fix wrong timing in sub, just load the video and the .ass or .srt file into aegisub, then under the timing tab, use shifting timing, it's base second, choose forward/backward. Done
  • To make transparent background, use RGBa color
